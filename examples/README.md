Examples
========

## How to Run Example

- Get source code.
```sh
$ git clone https://github.com/nakamura-to/KageDB
```

- Run server.
```sh
$ cd KageDB/example
$ node server.js
```

- Access with your browser.
```
http://localhost:9001/todo.html
```

This example bases on http://www.html5rocks.com/en/tutorials/indexeddb/todo/.
