Test
====

## How to Run Test

- Get source code.
```sh
$ git clone https://github.com/nakamura-to/KageDB
```

- Run server.
```sh
$ cd KageDB/test
$ node server.js
```

- Access with your browser.
 - For IE: `http://localhost:9000/ms.html`
 - For Firefox: `http://localhost:9000/moz.html`
 - For Chrome: `http://localhost:9000/webkit.html`
